Roadmap

So after spending many hours researching the best way to learn, as a visual learner, I decided to go with Udemy courses. These are high-quality video courses that I can learn from experienced tutors.

The drawback of this learning approach is the risk of falling into "tutorial hell," where students follow along with tutorials but fail to actively apply the knowledge they’ve gained in practical, real-world scenarios. I plan to overcome this by creating my own projects based on what I’ve learned or enhancing the course projects with additional features and functionality.

After checking out many tutors on udemy, I started to try out the two most popular tutors for web development, Dr. Angela and Colt Steele.

Phase 1: The Complete 2024 Web Development Bootcamp - Dr. Angela Yu (0–300 Hours)

Phase 2: The Web Developer BootCamp 2024 - Colt Steele (300–600 Hours)

Phase 3: The HTML & CSS Bootcamp - Colt Steele (500–700 Hours)

Phase 4: The Modern Javascript Bootcamp Course - Colt Steele (700–1000 Hours)

Phase 6: The Modern React Bootcamp (Hooks, Context, NextJS, Router) - Colt Steele (1000+ Hours)
